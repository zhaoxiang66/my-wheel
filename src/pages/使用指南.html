使用指南-介绍篇

一、框架背景

此框架是我在公司的项目中自己亲身实践，并且不想用一些别人繁琐的框架，而自己造的轮子,其实我也是在不断摸索中前进，通常我们项目中，尤其是做前台页面，用到最多的也就是轮播、分页、一些提示信息等，后台就很多了，比如单选框，按钮，下拉框，树形菜单，弹出框，提示信息等，我今后会一点一点往上加功能，争取做一个功能很全的一个js库！

二、框架依赖

此框架依赖jQuery！所以若使用此框架，必须要先引入jQuery，为什么要依赖jQuery呢？主要是它方便查询，它的一些方法非常好用，当然了，我也可以用原生的写，但是如果我们开发不用前端框架三巨头的话，免不了使用jQuery吧，所以我就选择jQuery当作辅助框架！ 下面是jQuery的一个CDN链接(举个例子)
< script src = "https://cdn.bootcss.com/jquery/3.3.1/jquery.min.js"><  /script>
如果想要更多版本，看这里
https://www.bootcdn.cn/jquery/

三、引入本框架

< script src = "https://cdn.bootcss.com/jquery/3.3.1/jquery.min.js"><  /script>
< script src = "zx-wheel.js"><  /script>
或者还可以引入
< script src = "zx-wheel.min.js"><  /script>
当然还要引入css文件
< link rel="stylesheet" href="zx-wheel.css">
或者还可以引入
< link rel="stylesheet" href="zx-wheel.min.css">

四、浏览器兼容性

关于浏览器兼容性，兼容谷歌内核，火狐，以及IE9+，如果我说兼容IE9+，大家应该知道什么意思了吧，当然是大部分浏览器都支持啦，IE8这种有内涵的浏览器洒家就不考虑了！！
